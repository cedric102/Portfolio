APP: Gravity in 2D ( Orbit Observation ).
- Project: Implement a 2D Gravity Graphical Application.
- Platform: Android Studio.
- Language: Java.
- Particles are generated and are in orbit around one gravity point.
- Each particles have their own initial Velocity and initial Angle.
- The particles do not interact with each other.
- There is no Colision nor mutual attraction.

Localisation Robot
Robotics: Robot Localisation.
- Project: Perform Robot Localisation Algorithm.
- Platform: Android Studio.
- Language: Java.
- Demonstrate Robot Localisation Algorithm.
- The blue robot needs to know its location in a given map.
- Particles are being generated with random angle / position.
- Simulated Light sensor is used as a way to determine the localisation on a map with markup on the floor.
- The blue robot's measurements are compared with the measurement of all the particles ( The red particles ).
- Then a particle selection of the closest match is performed.
Delta Robot
Robotics: Implement a Delta Robot.
- Project: Implement a Delta Robot Inverse Kynematis.
- Platform: Arduino.
- Language: C++.
Delta Robot is mostly used as a Robotic Arm which works as follows:
- The objective is to control the position of a point 'A'.
- Three servo motors ( Or other motors if required ) are staticly mounted to a frame.
- Each motors are connected in a similar way to point 'A', refer to Icon.
- The combination of motor angles control the position point 'A'.
